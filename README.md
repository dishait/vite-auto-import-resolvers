# node-lib-template

开箱即用的 node 库模板

<br />

## features

- pnpm 的
- vitest 测试
- 开箱即用的
- typescript 的

<br />

## Usage

### install

```shell
pnpm i
```

### test

```shell
pnpm test

// or pnpm test:watch
```

### build

```shell
pnpm build

// pnpm build:watch
```

### coverage

```shell
pnpm coverage
```

### dev

```shell
pnpm dev
```

### publish

```shell
npm publish
```

### play

```shell
pnpm play
```
